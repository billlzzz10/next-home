PROTOCOL ESSENTIALS:
====================

Request Types:
  - query: User sends message (stream SSE response)
  - settings: Bot configuration (return JSON)
  - report_reaction: User reacts (log it)
  - report_error: Error from Poe (acknowledge)

Response Events:
  - meta (required, first): Configure response
  - text (required, >=1): Response content
  - done (required, last): Complete response
  - error (optional): Error occurred
  - suggested_reply (optional): Follow-up suggestions
  - file (optional): Attachments

Limits:
  - Initial response: <= 5 seconds
  - Total response: <= 3600 seconds
  - Response size: <= 512,000 characters
  - Max events: <= 10,000
